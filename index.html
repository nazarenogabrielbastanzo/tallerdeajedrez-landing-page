Intro

¡Bienvenidos a mi sitio web, Taller de Ajedrez! Mi nombre es Nazareno Gabriel Bastanzo y me gusta mucho el Ajedrez, por lo que, desde el año 2014, intento brindar un sitio web completo, dedicado a este juego-ciencia. ¡Espero que disfruten de este sitio!

Work

Taller de Ajedrez fue construido desde sus comienzos, alrededor del año 2014, con la finalidad de fusionar dos ingredientes muy importantes para que sea el sitio que hoy es. Esos ingredientes son: La Programación y, por supuesto, el Ajedrez.

Es un proyecto personal, desarrollado desde el año 2014 hasta la actualidad. En este se han, y se siguen utilizando distintas tecnologías: Primero HTML, CSS y JavaScript. Luego PHP y MySQL. Ahora Angular + Firebase.

About

Taller de Ajedrez es un sitio web construido para el ajedrecista, pero también para todos los que les guste este hermoso juego.

Primeramente se quería brindar sólo algunas partidas de interés.
Luego se fueron agregando los demás contenidos del sitio (Frases, Fotos, Estudios, etc..)

Y hoy pretendo también, que quien esté interesado en el Ajedrez, y quiera dar los primeros pasos, no dude en ponerse en contacto conmigo.
Pueden hacerlo a través de las redes sociales que están más abajo, en esta misma página. ¡Nos vemos!
